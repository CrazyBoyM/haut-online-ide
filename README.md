# 在线代码编辑器（onlineIDE）  
## 后端
judge0 docker  
## 前端
vue3 + vite + typescript + tailwind + ace editor  
本项目主要为前端部分，后端请自行部署judge0 docker。  
参考：https://my.oschina.net/suncf/blog/4257003

## 功能
支持若干种语言（主要是judge0给力）  
支持ctrl + s 本地保存代码，刷新浏览器上次写的代码还在。  
支持ctrl + r 一键运行代码。
